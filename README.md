# redeman
Project Server for Redeman V2MR

## Como fazer o deploy para o Github

Para enviar suas alterações para o GitHub, execute o seguinte comando no terminal **de dentro deste diretório (`redeman`)**:

```bash
bash deploy.sh
```

Você também pode adicionar uma mensagem de commit personalizada:

```bash
bash deploy.sh "Sua mensagem de commit aqui"
```
